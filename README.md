# Line Bangkok PM2.5 Notification Bot
ใน repo นี้ผมขอนำเสนอโปรเจคบอทแจ้งเตือนค่า PM2.5 ในกรุงเทพมหานครขที่ผมได้ทำการศึกษาและลงมือทำขึ้นโดยในส่วนของการ coding ผมได้ใช้ภาษา JavaScript
(ในส่วนของค่า PM2.5 ในกทม.ได้ใช้ api ของทาง IQAir ในการส่งค่าและแสดงค่า PM2.5 ในกทม.)
ทั้งนี้ก่อนที่จะนำโค้ดที่ผมเขียนขึ้นไปใช้ทุกคนต้องศึกษาการสร้างบอทจากเว็บไซต์ Line developer ก่อน และจากนั้นนำโค้ดไป deploy ใน Google app script 
และใช้ trigger ของทาง Google app script ในการกำหนดเวลาการแจ้งเตือนของตัวบอทเพียงเท่านี้เราก็มีบอทให้ใช้ในการแจ้งเตือนค่า PM2.5 ได้แล้ว
